Описание проекта

Проект реализует AI-систему поиска информации по документам и внутренней базе знаний. Пользователь задаёт вопрос, система находит релевантные документы, после чего LLM формирует итоговый ответ.

Основные возможности

  • Semantic Search
  • Retrieval Augmented Generation
  • OpenAI integration
  • LangChain pipelines
  • Embeddings support
  • Vector Database
