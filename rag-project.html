Описание проекта

Проект реализует AI-систему поиска информации по документам и внутренней базе знаний. Пользователь задаёт вопрос, а AI находит релевантные данные и формирует ответ с помощью LLM.

Возможности

  • Retrieval Augmented Generation
  • Интеграция с OpenAI API
  • Semantic Search
  • Работа с документами
  • Vector Database
